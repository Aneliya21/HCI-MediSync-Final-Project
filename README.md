# MediSync

### Universal Digital Health Assistant

MediSync is a user-centered healthcare platform designed to simplify medication management, health monitoring, and communication between patients, caregivers, and healthcare professionals. Developed as a Human-Computer Interaction (HCI) project, the application focuses on accessibility, usability, and intuitive user experience.

---

## Overview

Healthcare applications often present challenges for elderly users and individuals with limited digital literacy. MediSync addresses these challenges by providing a simple and accessible interface that supports users in managing their daily healthcare activities.

The project demonstrates the application of Human-Computer Interaction principles, accessibility guidelines, and user-centered design methodologies in the healthcare domain.

### Key Objectives

* Improve medication adherence
* Enhance communication between patients and healthcare providers
* Support caregivers in monitoring patient activities
* Provide an accessible interface for users with different needs
* Reduce complexity in managing personal healthcare information

---

## Features

### Multi-Role User System

The platform supports different user roles:

* Patient
* Caregiver
* Healthcare Professional

Each role is designed around specific user needs and responsibilities.

### Secure Authentication Flow

* User registration
* Login functionality
* Form validation
* Password strength indication
* Email verification checks

### Accessibility First Design

MediSync includes an accessibility mode that improves usability for users who may require additional support.

Features include:

* Enhanced readability
* Simplified navigation
* Larger interactive elements
* Reduced cognitive load

### Medication Management

Users can:

* Add medications
* Track treatment schedules
* Manage medication information
* Simulate medication recognition through camera-based OCR workflows

### Personalized Dashboard

The dashboard provides:

* Quick access to essential information
* Personalized user experience
* Centralized healthcare management
* Intuitive navigation between application modules
